tipsyflix
=========

##Set up the application

    npm install -g yo
    npm install -g bower
    npm install
    cd client
    npm install
    bower install
    
## Run the application

###Server side
To run the node server app:

    node app.js
    
If you want auto-reload:

    npm install -g supervisor
    supervisor app.js

### Client side
To run the client development server:
    
    cd client
    grunt server
